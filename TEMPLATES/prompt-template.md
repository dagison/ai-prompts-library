# Prompt Template

This template helps structure prompts for better AI responses.

---

## Basic Prompt Structure

Role:
Act as a [expert role].

Context:
Provide background information about the task.

Task:
Clearly explain what needs to be done.

Format:
Specify how the output should be structured.

Constraints:
Define limits such as word count, tone, or style.

---

## Example

Role:
Act as a senior Python developer.

Context:
I am building a web scraper.

Task:
Write a Python script that scrapes article titles from a news website.

Format:
Return only the Python code.

Constraints:
Use requests and BeautifulSoup.

---

## Prompt Formula

Role + Context + Task + Format + Constraints

Example:

Act as a marketing strategist.

Create a 5-post Instagram content plan for a coffee shop.

Format:
- Post title
- Caption
- Hashtags

Tone: fun and engaging.
