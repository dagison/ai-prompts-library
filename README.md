# AI Prompts Library

A curated collection of powerful prompts for AI tools, LLMs, and automation workflows.

This repository is designed to help developers, creators, and entrepreneurs get better results from AI.

## Categories

- Coding Prompts
- Business & Startup Prompts
- Marketing Prompts
- Productivity Prompts
- AI Automation Prompts

## Example Prompt

### Startup Idea Generator

Prompt:

Act as a startup advisor.

Generate 10 startup ideas for the following niche:

Niche: [insert niche]

For each idea include:
- Problem
- Solution
- Target market
- Monetization strategy
