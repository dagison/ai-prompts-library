# How to Write Effective AI Prompts

Prompt writing is one of the most important skills when working with AI tools.

## 1. Start with the Objective

Clearly state what you want.

Example:
Write a LinkedIn post about AI automation for startups.

## 2. Define the Role

Assign a role to the AI.

Example:
Act as a startup advisor.

## 3. Provide Context

Example:

Our startup builds AI tools for small businesses.

Create a product launch announcement.

## 4. Specify Format

Example:

Write the output as:

- Title
- Introduction
- 3 key points
- Conclusion

## 5. Control Tone and Style

Examples:

Tone: professional  
Tone: casual  
Tone: persuasive  

Example prompt:

Write a persuasive email promoting a new AI tool for entrepreneurs.

## 6. Define the Length

Example:

Write a 300-word blog post.

## 7. Use Iterative Prompting

Improve results by refining prompts.

Example:

Make this explanation simpler.

or

Add real-world examples.

## 8. Use Constraints

Constraints help the AI focus.

Example:

Write a product description in under 100 words.
